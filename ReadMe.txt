To run this program you need to make use of the makefile,  and the
"Classifier.py" python file


NB: TO RUN PROGRAM PLEASE FOLLOW EXACTLY AS STEPS LISTED BELOW

You will need to create a virtual environment in order to use this project. In order to 
do that. Use the following commands:
make clean
make
source ./venv/bin/activate
make
python3 classifier.py
deactivate


Files of the program:
1) Classifier.py - this is the driver file for Classifier and has all the logic and models for the program 
3) Makefile - for building virtual environment
4) requirements.txt - requirements for virtual environment
